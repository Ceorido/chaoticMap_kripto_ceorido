Cara menjalankan Python proyek UAS SBOX Chaotic Sequence

1. Extract File ke Folder PC
2. instal streamlit dalam PC
3. setelah instal streamlit jalankan file app.py didalam folder "evaluation"
4. buka command prompt
5. masuk ke path folder tersebut "sbox-chaos-project-ceorido"
4. command untuk menjalankannya adalah "python -m streamlit run evaluation/app.py" atau apabila
sudah di folder evaluation jalankan "python -m streamlit run app.py"
5. Jika tidak bisa jalankan satu command lagi yaitu "streamlit run evaluation/app.py" atau
apabila sudah didalam folder "Evaluation" jalankan "streamlit run app.py"
6. Otomatis membuka streamlit di browser dengan membuka localhost
7. Terdapat 2 pilihan untuk menguji analisa sbox nya yaitu "Upload File" dan "generate Chaotic"
8. Apabila ingin menguji upload file ada di folder "Output" yaitu "sbox_optimized.txt" lalu klik
"lakukan analisa"
9. Apabila ingin generate Chaotic otomatis cukup klik "lakukan analisa"
10. Hasil akan langsung keluar semua termasuk plot graph visualization nya
